# Smart Tree

Smart Forest Alert adalah sistem deteksi dini kemiringan pohon berbasis Internet of Things (IoT) yang dikembangkan sebagai proyek skripsi.

## Fitur
- Monitoring sudut kemiringan pohon secara realtime
- Monitoring kelembaban tanah
- Monitoring kondisi hujan
- Klasifikasi status Normal, Waspada, dan Bahaya
- Notifikasi pada aplikasi mobile
- Penyimpanan data realtime menggunakan Firebase

## Teknologi
- ESP32
- MPU6050
- Soil Moisture Sensor
- Rain Drop Sensor
- Firebase Realtime Database
- React Native
- Expo Go

## Cara Kerja
Sensor membaca kondisi lingkungan dan kemiringan pohon, kemudian data dikirim ke Firebase. Aplikasi mobile akan menampilkan data secara realtime dan memberikan notifikasi ketika kondisi berbahaya terdeteksi.
